# Windows terminal hotkey
“in Linux you can open the terminal with ctrl / strg + Alt + T and This script will bring it also to Windows”
## how to use
first you will need download <a href="https://www.autohotkey.com/">AutoHotKey<a> and the <a href="https://github.com/EtwasPhoenix/Windows-terminal-hotkey/releases/tag/1.0">script <a>
if you want to have it start at startup you will need to put the script here `C:\Users\User\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup` to get there you can also just type `shell:startup`
